Сайт "Українська традиція" - це онлайн-колекція автентичних рецептів традиційної української кухні. Проект зберігає та передає кулінарну спадщину України, пропонуючи рецепти страв, які готували наші предки. Сайт міститиме рецепти різних регіонів України: борщ, вареники, голубці, сало, домашню ковбасу, паски, коржики та інші національні страви. Головна мета проекту - зберегти українські кулінарні традиції для майбутніх поколінь та допомогти людям готувати справжні українські страви вдома. 

Застосовані технології адаптивності:
- Flexbox - для header layout, навігації, card meta інформації
- CSS Grid - для адаптивної сітки рецептів (auto-fill, minmax)
- Медіазапити - адаптивний дизайн для tablet (≤768px) та mobile (≤480px)
- CSS модульна організація - розділення стилів на base.css, layout.css, components.css з використанням @import

JavaScript функціонал:
- DOM маніпуляція - зміна стилів recipe-card при наведенні, динамічне додавання елементів
- Динамічний контент - автоматичне відображення поточної дати у футері
- Акордеон - кнопка "Показати більше" для розгортання прихованого контенту
- Перемикач темної/світлої теми - збереження вибору в localStorage
- Інтерактивна навігація - підсвітка пунктів меню при наведенні (JS класами)
- Керування шрифтом - зміна розміру тексту за допомогою клавіш ArrowUp/ArrowDown
- Валідація форми - перевірка полів (ім'я мін. 3 символи, email з @ та доменом, повідомлення мін. 10 символів)
- LocalStorage - збереження теми сайту та імені користувача
